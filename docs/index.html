Features

Everything you need, nothing you don't

Full hardware-level control over every external monitor, with smart automation built in.

Per-Display Control

Independent brightness, contrast, and volume for every monitor via DDC/CI. No software dimming — real hardware-level adjustment.

Presets

Save and apply lighting configurations with one click. Create presets for work, movies, gaming, or night mode and switch instantly.

Auto Schedule

Set brightness rules for morning, evening, and night. Nit adjusts your displays automatically so you never have to think about it.

Dark Mode Sync

Automatically dims displays when your system switches to dark mode. Your eyes will thank you during late-night sessions.

F1/F2 + Hotkeys

Use your keyboard's function keys or set custom shortcuts to adjust brightness, contrast, and volume without touching your mouse.

Cross-Platform

macOS menu bar, Windows system tray, Linux via ddcutil. One app for every operating system — consistently designed and fully native.
